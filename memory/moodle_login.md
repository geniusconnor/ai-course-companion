**登录入口：** https://moodle.trine.edu

**流程：**
1. 点击 "Log in" → 跳转 Microsoft SSO
2. 输入邮箱（Connor 的 Trine 学生邮箱）
3. 输入密码
4. 触发 SMS 验证码 → **停下来，等 Connor 提供验证码**
5. 输入验证码 → 进入 Moodle 主页

**注意：**
- 密码页面有时不自动提交，需手动点"登录"按钮
- 每次新会话都需要重新登录
- MFA 验证码 Connor 会直接发在对话里（6位数字）
