## 使用指南

### 查询课程推荐教师
在对应专业的课程评价文件中，按课号筛选"推荐"列为"推荐"的记录。常见推荐规律：
- 推荐：写了就满分 / 给分松 / 事少 / 回邮件快
- 不推荐：查AI严格 / 给分低 / 无故扣分 / 作业量大且给分不对等

### 查询拼车同行
在对应校区的拼车文件中，按 Onsite 日期筛选，联系微信或手机号。

### 查询 CPT 雇主
在 [06_cpt_employers.md](06_cpt_employers.md) 中查看各雇主 CPT 支持情况。已知**不支持 CPT** 的雇主包括：EY、Silicon Inc、SoFi、Newsbreak、OpenAI、Coinbase。
