# EBSCO 登录流程

1. 打开 Moodle：https://moodle.trine.edu
2. Microsoft SSO 登录（邮箱 + SMS 验证码，Connor 提供）
3. 进入对应课程页面，点击阅读材料链接
4. 链接会自动跳转到 EBSCO 代理页面（已绑定 Trine 账号）
5. 在 EBSCO 页面找"Download PDF"或"Save"按钮

# 直接 PDF URL 提取方法

EBSCO PDF 查看器 URL 格式：
`https://research-ebsco-com.tuproxy.palni.edu/c/qwcd3d/viewer/pdf/{ID}`

尝试将 URL 末尾改为直接 PDF 下载链接：
`https://research-ebsco-com.tuproxy.palni.edu/c/qwcd3d/download/pdf/{ID}`

或在 EBSCO 页面顶部工具栏找下载图标（通常是向下箭头）。
